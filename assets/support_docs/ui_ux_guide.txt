===========================================
ACCESSIBILITY
===========================================

Focus States
- All interactive elements have focus states
- Focus border color: #4CAF50
- No outline removal

Required Field Indicators
- Visual asterisk (*)
- Required attribute on inputs

Label Association
- All inputs have associated labels
- For/ID attribute matching

===========================================
INTERACTIONS
===========================================

Discount Code Flow
1. User enters code in text field
2. Clicks "Apply" button
3. System validates code
4. Shows success or error message
5. Updates order summary if valid

Form Submission Flow
1. User fills required fields
2. Optionally applies discount
3. Selects payment method
4. Clicks "Place Order"
5. Form validates all fields
6. Generates order ID
7. Shows success message
8. Disables form

Payment Method Selection
1. User clicks payment method card
2. Card highlights with green border
3. Radio button auto-selects
4. Previously selected method deselects

===========================================
ANIMATION & TRANSITIONS
===========================================

Hover Effects
- Button background color transitions
- Smooth color changes

Focus Effects
- Border color change on input focus
- No jarring transitions

State Changes
- Payment method selection visual feedback
- Message display/hide transitions
